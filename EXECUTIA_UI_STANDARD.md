# EXECUTIA UI STANDARD — LOCKED

This file defines the visual and structural rules for executia.io.

## Core principle
EXECUTIA is a system interface, not a marketing website.

## Header — locked
- Left: `EXECUTIA™` + `EXECUTION STANDARD`
- Center: `ENTRY / GLOBAL / INSTITUTIONAL / REQUEST`
- Right: `Enter engine ↗`
- `EXECUTIA™` always links to `/`.
- Do not remove GLOBAL or INSTITUTIONAL.
- Do not replace the navigation with `ENTRY / ENGINE / REQUEST`.
- Navigation is intentionally quiet: low opacity, uppercase, small, institutional.

## Funnel
The site must always guide:

ENTRY → EXECUTION LAYER → REQUEST

## Color rules
- Never use black background.
- Background: white or very light blue only.
- Structural tone: dark blue `#0f2d4a` / `#1e3a5f`.
- Surfaces: `#f6f9fd`.
- Border: `#e3ebf3`.
- Green only for live/status confirmation.
- Red only for error/risk states.

## Typography
- One font family: Inter.
- Hero H1: 48–78px, weight 600.
- Section H2: 26–34px, weight 500.
- Body: 15–18px, weight 400.
- Nav: 11px, uppercase, quiet.
- Footer: 12px, compact.

## Spacing
Use only this spacing logic:
8 / 12 / 16 / 18 / 24 / 28 / 42 / 56 / 72 / 76

Avoid random spacing and page-to-page drift.

## Component rules
- Radius: 12px.
- Cards: 1px border, light surface, minimal shadow.
- Buttons: pill style, controlled, not oversized.
- Footer: compact, one line on desktop.

## Content rule
Homepage must have one dominant message:
Execution requires control.

No competing oversized slogans.
No heavy marketing blocks below the hero.
No fake execution statuses.


## Locked hero and footer rule
- Homepage primary statement is locked: "Execution is already happening. Control is missing."
- "Control is missing." may use a restrained critical accent; never use black background.
- Footer must use the original EXECUTIA column footer: brand / System / Structure / Contact / bottom line.


## HEADER BRAND LOCK — 2026-05-04
- EXECUTIA™ is the brand layer.
- EXECUTION STANDARD is the second layer, lighter and secondary.
- Clicking EXECUTIA™ always returns to Entry (`/` on public site, `https://executia.io` from engine).
- Header background is slightly transparent white, never black.
- Primary navigation is quiet by default; active `ENTRY` may be slightly stronger to preserve the flow.
- Header CTA `Enter engine ↗` is visible but not dominant.
- Buttons must not dominate the system interface; flow cues may be slightly highlighted only for `Entry ↗` and `Enter engine ↗`.
