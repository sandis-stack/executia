# EXECUTIA DESIGN SYSTEM

## Locked visual identity
EXECUTIA uses a light institutional system interface.

## Background rule
Never use black or near-black backgrounds.
Allowed backgrounds:
- `#ffffff`
- `#f6f9fd`
- `#eef5fb`

## Structural colors
- Primary ink: `#0f2d4a`
- Structural blue: `#1e3a5f`
- Muted text: `#6f8296`
- Border: `#e3ebf3`
- Status green: `#18704f`
- Risk red: `#9a3e3e`

## Typography
- Font: Inter
- Hero H1: 48–78px / 600
- Section H2: 26–34px / 500
- Body: 15–18px / 400
- Navigation: 11px / uppercase / quiet
- Footer: 12px / compact

## UI rules
- Radius: 12px
- Cards: light border, minimal shadow
- Buttons: pill CTA, controlled size
- Header and footer are not decorative; they are part of the EXECUTIA protocol.

## Locked header
`EXECUTIA®  EXECUTION STANDARD` + `ENTRY / ENGINE / ONE / DOCS / INSTITUTIONAL / PILOT` + `Enter Execution Engine ↗`

## Locked footer
Compact footer with EXECUTIA®, Entry, Execution Engine, EXECUTIA® ONE, Documentation, Institutional, Pilot, Global, Contact and copyright.
